What is Tix?

Tix is a programming library for the TK toolkit. A Brief description of the Tix library is in the file ABOUT.html.

About This File

This file is the master index of all the off-line documentation included in this package. For the on-line documentation of Tix served in the World Wide Web, please visit the Tix Home Page at http://www.xpi.com/tix/ .

The file README.html is in HTML format and can be read in a HTML browser. If you do not have an HTML browser, each HTML document included in this package has a plain text version. For example, the plain text version of README.html is README.txt.
