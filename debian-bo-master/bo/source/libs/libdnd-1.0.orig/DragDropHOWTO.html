The Drag and Drop HOWTO

César Augusto Rorato Crusius, c.crusius@ieee.org

v1.0, 5 February 1996

This document describes how to enable your X and Xt programs to use drag and drop using the DND protocol. A C Xt interface to DND is provided with OffiX, a collection of general utilities already using DND protocol (file manager and text editor included).
