
MORE SPECIAL BOOT PARAMETERS

You can use the following boot parameters at the `boot:' prompt.
If you use hex numbers you have to use the 0x prefix (e.g. 0x300)

protect I/O port regions     reserve=iobase,extent[,...]
Adaptec 151x, 152x           aha152x=iobase[,irq[,scsi-id[,reconnect]]]
Adaptec 1542                 aha1542=iobase[,buson,busoff[,dmaspeed]]
Adaptec 274x, 284x           aic7xxx=no_reset (enabled if non-zero)
BusLogic SCSI Hosts          buslogic=iobase
Future Domain TMC-8xx/950    tmc8xx=mem_base,irq
Pro Audio Spectrum           pas16=iobase,irq
Seagate ST-0x                st0x=mem_base,irq
Trantor T128                 t128=mem_base,irq
ATAPI/IDE CD-ROM             hd{a,b,c,d}=cdrom   (usually unnecessary)

Modules:
Ethernet device drivers and drivers for the proprietary CD-ROM
interfaces are loaded as modules, so you cannot give paramters for
them at the boot prompt.
