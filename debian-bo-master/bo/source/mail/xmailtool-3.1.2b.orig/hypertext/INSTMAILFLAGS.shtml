INSTMAILFLAGS

On some systems, a the MailBox Direcotry is writable only by processes in a special mail group. In order for XMailTool to create lock files in such a directory, it will need to have SETGID permissions and belong to that group. The INSTMAILFLAGS is used durring the install step to set the correct owner, group and permissions on the XMailTool executable (xmailtool).

EXAMPLE

     INSTMAILFLAGS = -g mail -m 2755
XMailTool will be SETGID to group mail. 

     INSTMAILFLAGS =
XMailTool will have no special permissions or ownership settings.
