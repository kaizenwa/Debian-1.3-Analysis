Setting the debug Variable

The debug variable is used to indicate that XMailTool should print debugging information in the verbose output window and to stderr. If debug is set the temporary mailbox file will not be invisible and will stay around after XMailTool terminates.

Examples

set debug
set debug=1
set debug=true
set debug=yes
All of the above are equivalent and indicate that XMailTool should print debugging information. 
set debug=0
set debug=false
set debug=no
All of the above are equivalent and indicate that XMailTool should not print debugging information.

Default

set debug=0
