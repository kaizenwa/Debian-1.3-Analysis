SPECIAL_FLAGS

On some systems, there are additional flags needed beyond those defined by either Imake or any of the other Imake Configuration Parameters. If your system requires such a flag, you should specify them using SPECIAL_FLAGS.

EXAMPLE

     SPECIAL_FLAG = -U__STR__
This is required for IBM RS/6000. 

     SPECIAL_FLAG =
XMailTool will have no special flags when compiled.
