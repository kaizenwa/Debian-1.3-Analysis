Troubleshooting

The problems you may run into with XMailTool have been broken down into two categories. The Build Problems section talks about what you should do if you can't get XMailTool to build or install correctly. The User Configureation Problems section talks about things that may go wrong when running XMailTool.

If you have problems that aren't described in either of these sections, please feel free to send me email (bobo@cray.com).
