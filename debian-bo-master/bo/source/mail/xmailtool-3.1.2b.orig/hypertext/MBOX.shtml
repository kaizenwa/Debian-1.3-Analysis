Setting the MBOX Variable

The MBOX variable is used to indicate that name of the file XMailTool should use to save messages that are not to remain in the system mailbox, and are not marked as deleted. A message is to be removed from the system mailbox if the hold variable is not set, and the message has been read durring the current session of XMailTool. The MBOX variable also indicates the name of the file XMailTool should display in the File Name: window when XMailTool is first started. The value of MBOX will always be the first value in the Saved FNames List.

Examples

csh:

setenv MBOX "+mbox"

ksh/sh:

MBOX="~/mail/mbox"
export MBOX

.mailrc or .xmailtoolrc:

set MBOX="~/mailbox"

Default

"~/mbox"
