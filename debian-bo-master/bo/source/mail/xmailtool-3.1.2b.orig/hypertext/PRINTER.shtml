Setting the PRINTER Variable

The PRINTER variable is used to indicate that name of the printer the print command is sending data to. The value of this variable isn't actually used by XMailTool. Rather, XMailTool passes it to the print command. The significance and valid values for the PRINTER variable are determined by the print command being used and your site configuration.

Examples

csh:

setenv PRINTER "f610d"

ksh/sh:

PRINTER="f610d"
export PRINTER

.mailrc or .xmailtoolrc:

set PRINTER="f610d"

Default

The PRINTER variable has no default value. You should check with your system administrator to determin what will happen to your message if you print without specifying a PRINTER value.
