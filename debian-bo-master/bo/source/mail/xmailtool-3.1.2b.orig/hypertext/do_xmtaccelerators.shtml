Setting the do_xmtaccelerators Variable

The do_xmtaccelerators variable is used to indicate that XMailTool should install accelerator keys in each of the un-editable text widows of the main window. These accelerator keys can cause unusual and possibly irreversable problems if you accidentally start typing in one of these windows.

Examples

set do_xmtaccelerators
set do_xmtaccelerators=1
set do_xmtaccelerators=true
set do_xmtaccelerators=yes
All of the above are equivalent and indicate that XMailTool should install the accelerator keys. 
set do_xmtaccelerators=0
set do_xmtaccelerators=false
set do_xmtaccelerators=no
All of the above are equivalent and indicate that XMailTool should not install the accelerator keys.

Default

set do_xmtaccelerators=1
