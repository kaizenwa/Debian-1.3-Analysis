Using the set command

The set command is used in the mailrc file to associate a value with a variable or to set a boolean variable. The format of the set command is as follows: 
set variable[=['|"]value['|"]]
The variable name is case sensative. If the variable name is given without the =value clause, XMailTool will treat the variable as a boolean and set the corresponding value to 1, true, or yes. If value is given with quotes, the opening and closing quote types must match.

Examples

set hold
set alwaysignore=0
set bodyOnly=true
set replyto="bobo@cray.com"
set interval=15

Differences

XMailTool handles the set command slightly different than other mail readers. Only one variable name can be used with each set command. Therefore the following: 
set hold bodyOnly
would be incorrect. It should be written as 
set hold
set bodyOnly
XMailTool doesn't recognize the "no" prefix for boolean variables. If you want a boolean variable set to false, you must specify it as on of the following: 
set hold=0
set hold=false
set hold=no
