Setting the askcc Variable

The askcc variable is used to indicate that XMailTool should include a Cc: line in all outbound messages editor windows even if there aren't any appropriate addresses to be put on this line. The empty Cc: line is used to remind the user where to put Cc: addresses. If the Cc: line is empty when the message is sent, XMailTool removes it from the message header before passing it to the sendmail command.

Examples

set askcc
set askcc=1
set askcc=true
set askcc=yes
All of the above are equivalent and indicate that a Cc: line should be included. 
set askcc=0
set askcc=false
set askcc=no
All of the above are equivalent and indicate that a Cc: line shoule not be included.

Default

set askcc=0
