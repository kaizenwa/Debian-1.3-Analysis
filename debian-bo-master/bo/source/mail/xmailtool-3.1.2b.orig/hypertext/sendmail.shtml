Setting the sendmail Variable

The sendmail variable is used to specify the command used when sending a mail message. This string should be a command line that is valid for your value of $SHELL. The outbound message file will be redirected to this processes stdin. It is expected that this command will read its stdin. Any output from this command will be considerd as an Error message and will be displayed in XMailTools verbose output window. See the Environment Variables XMailTool Sets section to see what variables are set for external processes such as the sendmail.

Examples

set sendmail='$HOME/bin/expand_aliases | /usr/lib/sendmail -t'
set sendmail="$HOME/bin/my_mailer"

Default

The default value of sendmail is determined from the MSENDER build flag. Check with the person who built XMailTool to determine what this value is.
