SYS_LIBRARIES

Some systems require additional libraries than those specified by the XawClientLibs definition. These libraries may include the math library or memory management library. If you build XMailTool and the compiler complains about unresolved externals, it is possible that you need to specify additional libraries using the SYS_LIBRARIES variable.

EXAMPLE

   SYS_LIBRARIES = -lmath -lmalloc XFACELIB
Link XMailTool with the libmath.a and libmalloc.a and the XFace library. 

  SYS_LIBRARIES = -lm XFACELIB
On some machines, the math library is in libm.a instead of libmath.a.
