Setting the do_signature Variable

The do_signature variable is used to indicate that XMailTool should append the contents of the ~/.signature file to the end of each outbound message. This can be usefull if the sendmail command does this for you.

Examples

set do_signature
set do_signature=1
set do_signature=true
set do_signature=yes
All of the above are equivalent and indicate that XMailTool should append the contents of ~/.signature. 
set do_signature=0
set do_signature=false
set do_signature=no
All of the above are equivalent and indicate that XMailTool should not append the contents of ~/.signature.

Default

set do_signature=1
