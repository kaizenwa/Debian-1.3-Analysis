How to Get XMailTool

Since you already have network connectivity through HTTP, the easiest way to get XMailTool is to click here and select a location to put the file xmailtool.tar. This should be the most up to date version of XMailTool that is publicly available.

This is a list of other places to look:

  • ftp.x.org
  • gatekeeper.dec.com
  • ftp.khoros.unm.edu
