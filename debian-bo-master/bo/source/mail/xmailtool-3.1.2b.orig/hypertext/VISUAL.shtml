Setting the VISUAL Variable

The VISUAL variable is used to indicate the name of the editor used when the ~v command is given in the outbound or confirm windows. This is similar to the EDITOR variable except that it will only be executed after the internal editor has been executed. The edit_needs_xterm indicates if this command needs to be run within an xterm interface.

Examples

csh:

setenv VISUAL "ed"

ksh/sh:

VISUAL="emacs"
export VISUAL

.mailrc or .xmailtoolrc:

set VISUAL="xedit"

Default

"vi"
