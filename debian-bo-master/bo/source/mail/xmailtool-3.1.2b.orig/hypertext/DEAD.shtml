Setting the DEAD Variable

The DEAD variable is used to indicate the name of the file XMailTool should copy an outbound message to if the user has canceld delivery of that message.

Examples

csh:

setenv DEAD +dead.letter

ksh/sh:

DEAD=~/dead.letter
export DEAD

.mailrc or .xmailtoolrc:

set DEAD="/dev/null"

Default

~/dead.letter
