Environment Variables XMailTool Uses

XMailTool obtains the values for some variables from the environment if they aren't specified using X11 resources or in the .mailrc or ~/.xmailtoolrc files. These variables are as follows:
  • PRINTER
  • DEAD
  • EDITOR
  • VISUAL
  • SHELL
  • MBOX
  • MAIL
  • MAILRC
  • folder
  • TMPDIR

In csh, these variables are set using the setenv command.

Examples

setenv DEAD ~/dead.letter
setenv PRINTER "f610d"
In ksh or sh, these variables must be exported.

Examples

DEAD=~/dead.letter
PRINTER="f610d"
export DEAD PRINTER

Environment Variables XMailTool Sets

XMailTool sets the values of some variables in the environment for use by child processes. The value of these variables are taken from either the current value used by XMailTool, or from the current message. These variable are as follows:
  • DISPLAY
  • PRINTER
  • DEAD
  • EDITOR
  • VISUAL
  • SHELL
  • MBOX
  • MAIL
  • MAILRC
  • folder
  • TMPDIR
  • SUBJECT
  • FROM
  • DATE
  • KEYWORDS
  • COMMENTS
