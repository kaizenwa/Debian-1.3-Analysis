Setting the bodyOnly Variable

The bodyOnly variable is used to indicate that XMailTool should display only the body of a message. This variable controls the initial value of the BodyOnly toggle button. If bodyOnly is set, XMailTool will not display any header information.

Examples

set bodyOnly
set bodyOnly=1
set bodyOnly=true
set bodyOnly=yes
All of the above are equivalent and indicate that XMailTool should display only the body of each message. 
set bodyOnly=0
set bodyOnly=false
set bodyOnly=no
All of the above are equivalent and indicate that XMailTool should display both header and body of the message.

Default

set bodyOnly=0
