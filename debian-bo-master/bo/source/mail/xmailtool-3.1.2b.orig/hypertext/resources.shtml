XMailTool Resource Guide

Introduction

XMailTool is very dependant on X11 resources for much of its functionality. For this reason, each executable has a resource patchlevel string compiled into it. When XMailTool starts up, it checks the patchlevel found in the aplication defaults (app-defaults) resource file. If these two strings don't match, XMailTool will complain about this. See the XMailTool Doesn't Run section of the Runtime Problems FAQ for more details.
