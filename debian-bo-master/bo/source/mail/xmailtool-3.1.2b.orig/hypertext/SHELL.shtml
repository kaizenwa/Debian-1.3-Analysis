Setting the SHELL Variable

The SHELL variable is used to indicate the name of the shell used when executing external commands such as print_cmd, EDITOR, and sendmail.

Examples

csh:

setenv SHELL "/bin/csh"

ksh/sh:

SHELL="/bin/ksh"
export SHELL

.mailrc or .xmailtoolrc:

set SHELL="/usr/local/bin/ksh"

Default

/bin/sh
