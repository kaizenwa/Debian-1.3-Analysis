Setting the alwaysignore Variable

The alwaysignore variable is used to indicate that the ignore/retain list should be considerd when printing, saving, and copying a message to a file.

Examples

set alwaysignore
set alwaysignore=1
set alwaysignore=true
set alwaysignore=yes
All of the above are equivalent and indicate that the ignore/retain list should be considerd. 
set alwaysignore=0
set alwaysignore=false
set alwaysignore=no
All of the above are equivalent and indicate that the ignore/retain list should not be considerd.

Default

set alwaysignore=0
