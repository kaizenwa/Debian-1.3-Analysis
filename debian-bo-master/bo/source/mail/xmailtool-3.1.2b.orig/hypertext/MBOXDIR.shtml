MBOXDIR

The location of the system mailbox file differs greatly from one machine to the next and from one administration style to the next. The value of MBOXDIR is used to calculate where a users default system mailbox file is located. It is used in the following context: 
        sprintf(filename,"%s/%s",MBOXDIR,username);

EXAMPLE

        MBOXDIR = -DMAILBOX_DIRECTORY=\"/usr/mail\"


        MBOXDIR = -DMAILBOX_DIRECTORY=\"/var/spool/mail\"
