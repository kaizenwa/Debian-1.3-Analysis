HELP_DIR

The location of the directory that holds help files for X11 applications is determined by the administrator of a machine. This value is rarely specified by the manufacturer of the machine. HELP_DIR indcates where XMailTool should install its help file, and wehre it should look to find its help file.

EXAMPLE

        HELP_DIR = $(LIBDIR)
If $(LIBDIR) has been defined as /usr/lib/X11, this causes XMailTool to install xmailtool.help as /usr/lib/X11/xmailtool.help. 

        HELP_DIR = $(LIBDIR)/help
This causes XMailTool to install xmailtool.help as /usr/lib/X11/help/xmailtool.help.
