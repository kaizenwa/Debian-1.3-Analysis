Using the -interval Command Line Option and Resource

This command line option and resource is similar to the interval variable. It is used to specify the period of time in seconds XMailTool should wait between checking the MAIL file for new mail. Setting this to a low value will mean that XMailTool will check more frequently which may cause more load on your system. A high value will mean that XMailTool will check less frequently which means you may not be notified right away when new mail arrives. XMailTool is expecting an integer argument following this command line option and an integer value in the resource. The interval resource has a higher priority than the mailrc variable, and the command line option has a higher priority than the resource.

Examples

Command Line option

xmailtool -interval 15

Resource

XMailTool.interval: 25

Default

By default XMailTool will wait 15 seconds.
