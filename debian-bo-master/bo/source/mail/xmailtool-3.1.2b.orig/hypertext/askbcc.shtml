Setting the askbcc Variable

The askbcc variable is used to indicate that XMailTool should include a Bcc: line in all outbound messages editor windows even if there aren't any appropriate addresses to be put on this line. The empty Bcc: line is used to remind the user where to put Bcc: addresses. If the Bcc: line is empty when the message is sent, XMailTool removes it from the message header before passing it to the sendmail command.

Examples

set askbcc
set askbcc=1
set askbcc=true
set askbcc=yes
All of the above are equivalent and indicate that a Bcc: line should be included. 
set askbcc=0
set askbcc=false
set askbcc=no
All of the above are equivalent and indicate that a Bcc: line shoule not be included.

Default

set askbcc=0
