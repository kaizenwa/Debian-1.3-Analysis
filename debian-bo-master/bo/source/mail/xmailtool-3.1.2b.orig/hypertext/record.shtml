Setting the record Variable

The record variable is used to specify the path to the file in which XMailTool is to keep a copy of outgoing messages. If record isn't set, XMailTool will not keep a copy of outgoing messages. In addition to setting the record string, the do_xmtrecord variable needs to be set if you want XMailTool to do the recording. not be sensitive.

Examples

set record="~joe/outbound"
set record="outbound"
set record="+outbound"
set record="$HOME/outbound"

Default

record has no default value.
