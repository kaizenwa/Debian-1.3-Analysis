Using the -file Command Line Option and Resource

This command line option and resource is similar to the MAIL variable. It is used to indicate the name of the file XMailTool should use as the System MailBox file. XMailTool is expecting a string argument following this command line option and a string value in the resource. The file resource has a higher priority than the mailrc variable, and the command line option has a higher priority than the resource.

Examples

Command Line option

xmailtool -file +work

Resource

XMailTool.file: ~joe/mbox/stuff

Default

MBOXDIR/$username
