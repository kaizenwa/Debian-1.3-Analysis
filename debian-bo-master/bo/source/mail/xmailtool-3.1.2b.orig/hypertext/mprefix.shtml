Setting the mprefix Variable

The mprefix variable is used to specify the string that is prepended to each line when XMailTool quotes another message using either the reply or forward functions. This variable and the indentprefix variable are interchangeable.

Examples

set mprefix="> "
set mprefix=":-) "
set mprefix="You Said> "

Default

set mprefix="> "
