Setting the showdeleted Variable

The showdeleted variable is used to indicate that XMailTool should display message header lines even if they've been marked for deletion. If showdeleted isn't set, XMailTool will remove those lines from the header window. This variable controls the initial value of the Show Deleted toggle button.

Examples

set showdeleted
set showdeleted=1
set showdeleted=true
set showdeleted=yes
All of the above are equivalent and indicate that XMailTool should display messages that have been marked for deletion. 
set showdeleted=0
set showdeleted=false
set showdeleted=no
All of the above are equivalent and indicate that XMailTool should not display messages that have been marked for deletion.

Default

set showdeleted=0
