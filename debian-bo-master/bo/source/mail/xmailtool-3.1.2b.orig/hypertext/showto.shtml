Setting the showto Variable

The showto variable is used to control what address is displayd in the header window if the user sent the message. If showto is set, and a message is from the user, XMailTool will display the To: address list instead. This is particularly usefull if you are viewing your record file.

Examples

set showto
set showto=1
set showto=true
set showto=yes
All of the above are equivalent and indicate that XMailTool should display the To: address list if the messages is from the current user. 
set showto=0
set showto=false
set showto=no
All of the above are equivalent and indicate that XMailTool should not display the To: address list if the messages is from the current user.

Default

set showto=0
