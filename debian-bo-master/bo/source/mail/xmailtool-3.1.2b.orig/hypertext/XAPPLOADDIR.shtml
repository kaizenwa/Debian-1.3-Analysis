XAPPLOADDIR

The location of the directory that holds app-defaults files for X11 applications may very from one site to another. The value of XAPPLOADDIR indcates where XMailTool should install its app-defaults files.

EXAMPLE

       XAPPLOADDIR = $(LIBDIR)/app-defaults
If $(LIBDIR) has been defined as /usr/lib/X11, this causes XMailTool to install XMailTool.ad and XMailTool-color.ad as /usr/lib/X11/app-defaults/XMailTool and /usr/lib/X11/app-defaults/XMailTool-color.
