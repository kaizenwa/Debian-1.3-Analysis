Setting the replyto Variable

The replyto variable is used to specify the string XMailTool should put in the Reply-To: header line of outbound messages. This is used if your system doesn't specify this value correctly, or if you want to get replies at a different address than you used to send the original. If you don't specify a replyto value, XMailTool will not add this header line to outbound messages.

Examples

set replyto="joe@otherplace.net"
set replyto="joe@thishost.net"

Default

replyto has no default value.
