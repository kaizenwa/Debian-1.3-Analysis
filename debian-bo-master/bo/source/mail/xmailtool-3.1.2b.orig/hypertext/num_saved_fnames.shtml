Setting the num_saved_fnames Variable

The File Name entry window is implemented as a circular list of the most recently accessed file names. File names are added to the list when either the save or file commands are used. Names are removed from the list in FIFO style when the limit of the list is reached. Clicking the "File Name:" button causes the entry to circulate through the list. The list limit is specified by the user by setting the num_saved_fnames. $MBOX will always be the first element in the list. The file name list can be primed by setting the saved_fnames variable.

Examples

set num_saved_fnames=15
# not quite enough
set num_saved_fnames=1
# overkill
set num_saved_fnames=1000

Default

set num_saved_fnames=5
