Setting the saved_fnames Variable

The File Name entry window is implemented as a circular list of the most recently accessed file names. File names are added to the list when either the save or file commands are used. Names are removed from the list in FIFO style when the limit of the list is reached. Clicking the "File Name:" button causes the entry to circulate through the list. The saved_fnames variable is used to prime the list of file names. XMailTool will enter this list of names up to 1 less than num_saved_fnames.

Examples

set saved_fnames="~joe/mbox +outbox +work $HOME/stuff"
set saved_fnames="~joe/mbox, +outbox, +work $HOME/stuff"

Default

saved_fnames has no default value.
