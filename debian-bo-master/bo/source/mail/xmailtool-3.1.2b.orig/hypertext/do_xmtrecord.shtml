Setting the do_xmtrecord Variable

The do_xmtrecord variable is used to indicate that XMailTool should save a copy of each outbound message to the file specified in the record variable. This can be usefull if the sendmail command does this for you.

Examples

set do_xmtrecord
set do_xmtrecord=1
set do_xmtrecord=true
set do_xmtrecord=yes
All of the above are equivalent and indicate that XMailTool should save a copy of the outbound messages. 
set do_xmtrecord=0
set do_xmtrecord=false
set do_xmtrecord=no
All of the above are equivalent and indicate that XMailTool should not save a copy of the outbound messages.

Default

set do_xmtrecord=1
