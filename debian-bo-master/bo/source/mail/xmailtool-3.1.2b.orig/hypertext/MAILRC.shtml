Setting the MAILRC Variable

The MAILRC variable is used to indicate that name of the file XMailTool should read for aliases, variable settings, and ignore/retain list settings. This file is read every time the mailrc button is pressed.

Examples

csh:

setenv MAILRC "~/.mailrc"

ksh/sh:

MAILRC="~/mail/.mailrc"
export MAILRC

.mailrc or .xmailtoolrc:

set MAILRC="~/xmailtool_startup"

Default

"~/.mailrc"
