Make Configuration

A fairly generic Makefile has been provided with the source for XMailTool. It is called Makefile.std. This file was actually generated from Imake on Solaris 2.3 and then modified to make it more readable and less obtuse. Many of the steps for using Make are the same as for using Imake. There are some important paramameters you need to set in the Makefile. Before you start, you may want to make a copy of the original Makefile.std.

Copy Makefile.std to Makefile. You shouldn't need to edit the Makefiles in compface or utilities, but you will need top copy the Makefile.std in those directories to Makefile in the respective directory.

This is a list of the Makefile configuration parameters you may need to change:

  • HAS_XFACE
  • SYS_LIBRARIES
  • HELP_DIR
  • XAPPLOADDIR
  • MANDIR
  • BINDIR
  • HYPERTEXTDIR
  • MBOXDIR
  • MSENDER
  • PRINTER_CMD
  • NEED_DIRENT
  • NEED_TRUNCATE
  • NEED_MKSTEMP
  • NEED_STRSTR
  • NEED_CASECMP
  • NEED_STRERROR
  • NEED_SETENV
  • SPECIAL_FLAGS
  • INSTMAILFLAGS
  • MANSUFFIX
  • STD_DEFINES
  • HYPERTEXT
  • DEFURL
