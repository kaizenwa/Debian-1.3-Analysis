Setting the keep Variable

The keep variable is used to indicate that XMailTool should delete a mailbox file if it contains no (0) messages. This situation may occur if the user has removed all messages from this mailbox. If keep is set, XMailTool won't delete the mailbox file.

Examples

set keep
set keep=1
set keep=true
set keep=yes
All of the above are equivalent and indicate that XMailTool should not delete the empty mailbox file. 
set keep=0
set keep=false
set keep=no
All of the above are equivalent and indicate that XMailTool should delete the empty mailbox file.

Default

set keep=0
