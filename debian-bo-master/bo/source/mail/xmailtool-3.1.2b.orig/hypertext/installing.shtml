Installing XMailTool

Whether you have used Imake to create the Makefiles, or copied them from Makefile.std, all you should have to do at this point is type "make install install.man." You may want to use the -k option on the make command. If BINDIR, MANDIR, HELP_DIR, MANSUFFIX, and INSTMAILFLAGS have been set correctly, then xmailtool, its app-defaults files, its help file, and its man page should all have been copied to appropriate places on your system.
