DEFURL

If you need to use a local hyper text server instead of the one at Cray Research, Inc., then you need to change the DEFURL definition. This value will be passed to the HYPERTEXT command in the following context: 
sprintf(help_cmd,"%s %s",HYPERTEXT,DEFURL);
The help_cmd value is then passed to the SHELL command so it can contain additional command line options for the HYPERTEXT client. NOTE: The *.shtml files associated with XMailTools hyper text help makes use of many of the fancy features of the NCSA httpd 1.3 server. In order to take advantage of these features, you will need to run a server that is equivalent or better than NCSA httpd 1.3 server.

EXAMPLE

     DEFURL = -DDEFURL=\"http://poplar117.cray.com:3939/~bobo/XMailTool/XMailTool.shtml\"
This is the standard default setting. 

     DEFURL = -DDEFURL=\"http://localwww/XMailTool/XMailTool.shtml\"
This would use localwww instead of the machine at CRI.
