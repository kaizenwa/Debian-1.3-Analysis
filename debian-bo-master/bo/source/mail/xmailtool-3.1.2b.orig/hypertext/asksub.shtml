Setting the asksub Variable

The asksub variable is used to indicate that XMailTool should include a Subject: line in all outbound messages editor windows. The empty Subject: line is used to remind the user where to put the Subject: line. If the Subject: line is empty when the message is sent, XMailTool removes it from the message header before passing it to the sendmail command.

Examples

set asksub
set asksub=1
set asksub=true
set asksub=yes
All of the above are equivalent and indicate that a Subject: line should be included. 
set asksub=0
set asksub=false
set asksub=no
All of the above are equivalent and indicate that a Subject: line shoule not be included.

Default

set asksub=1
