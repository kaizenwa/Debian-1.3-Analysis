Setting the MAIL Variable

The MAIL variable is used to indicate the name of the file XMailTool should use as the System MailBox file. The state of this file is represented by the bitmap in the icon. The file is checked periodicly specified by the interval variable or the interval command line option and resource.

Examples

csh:

setenv MAIL +otherstuff

ksh/sh:

MAIL=~/project.mail
export MAIL

.mailrc or .xmailtoolrc:

set MAIL="~common/project.mail"

Default

MBOXDIR/$username
