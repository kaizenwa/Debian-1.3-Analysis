HYPERTEXT

The help feature for XMailTool is provided through an external program that retreives information from WWW style servers. Since you are reading this document, you probably already know of at least one such client. A common hypertext client is NCSA Mosaic. This is available for X Windows as well as Macintosh and PC/Windows. The value of this parameter is used for the default setting for the hypertext variable which can be set by the user. If The HYPERTEXT flag isn't set when XMailTool is built, XMailTool will use the primative built in version of help (not recomended).

EXAMPLE

      HYPERTEXT = -DHYPERTEXT=\"Mosaic\"
This will cause XMailTool to call NCSA Mosaic when the help button is pressed. 
      HYPERTEXT =
This will cause XMailTool to display a very breif help message.
