Setting the EDITOR Variable

The EDITOR variable is used to indicate the name of the editor used when composing outbound messages. If the value is "none" or is not set in either the .mailrc or .xmailtoolrc files or environment, XMailTool will use an internal editor which is built on top of the Xaw Text Widget.

Examples

csh:

setenv EDITOR "emacs"

ksh/sh:

EDITOR="vi"
export EDITOR

.mailrc or .xmailtoolrc:

set EDITOR="none"

Default

"none"
