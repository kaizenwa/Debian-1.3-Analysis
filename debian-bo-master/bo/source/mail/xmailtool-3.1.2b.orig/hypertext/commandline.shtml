Command Line Options

Some of XMailTools behavior can be controlled by command line options. Most of XMailTools command line options are actually interpreted by libXt.
If someone knows a good link to libXt command line options documentation, I would like to put it here.
After libXt has extracted and interpreted the command line options that it understands, XMailTool interprets the rest. It is my goal that eventually more of XMailTools behavior will be controllable from command line options and resources, but for now it is very limited. Below is a list of the command line options that XMailTool recognizes.
  • {+|-}autoHelp
  • {+|-}cmdBox
  • {+|-}stateBox
  • -interval value
  • -file value
  • -iconGeometry value
  • # value
  • -font value
