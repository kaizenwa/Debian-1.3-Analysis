Setting the detectiv Variable

The detectiv variable is used to indicate that XMailTool should be extra picky about certain problem conditions. If deteciv is set, XMailTool will complain about messages that don't have From: or To: header lines, and messages whose Content-Length: and X-Lines: values are incorrect. If detectiv isn't set, XMailTool will ignore these minor problems and adjust the Content-Length: or X-Lines: values.

Examples

set detectiv
set detectiv=1
set detectiv=true
set detectiv=yes
All of the above are equivalent and indicate that XMailTool should be extra picky. 
set detectiv=0
set detectiv=false
set detectiv=no
All of the above are equivalent and indicate that XMailTool should not be extra picky.

Default

set detectiv=0
