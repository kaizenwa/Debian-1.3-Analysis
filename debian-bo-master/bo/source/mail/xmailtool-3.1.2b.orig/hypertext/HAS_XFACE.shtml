HAS_XFACE

The support for X-Face header lines is provided through the compface library. The HAS_XFACE variable controls weather XMailTool will recognize X-FAce header lines. If you do not wish to use X-Face header lines or do not want XMailTool to recognize X-Face header lines, or for some reason cann't get the compface library to build, you should not use -DHAS_XFACE. Note: compface is provided "as is." No support will be provided for problems using the compface library or utilities, and no documentation will be provided for porting or using the compface library or utilities.

EXAMPLE

   HAS_XFACE     = -DHAS_XFACE
XMailTool will create the xface widget and recognize X-Face header lines. 

   HAS_XFACE     =
XMailTool will not create the xface widget and will not recognize X-Face header lines.
