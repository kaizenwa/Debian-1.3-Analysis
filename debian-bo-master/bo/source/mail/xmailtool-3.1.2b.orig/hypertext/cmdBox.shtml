Using the {+|-}cmdBox Command Line Option and Resource

This command line option and resource is similar to the cmd_box variable. It indicates that XMailTool should use a Box widget to hold the command buttons in the tool window. Depending on your resources, you may be able to squeek all of the command buttons into two rows instead of three. This may save space while compromising the way the command buttons look. If you prefer to use the Athena Box widget, you should use the -cmdBox option or set the cmdBox resource to true. The cmdBox resource has a higher priority than the mailrc variable, and the command line option has a higher priority than the resource.

Examples

Command Line option

xmailtool -cmdBox

Resource

XMailTool.cmdBox: True

Default

By default XMailTool will use the Athena Form widget to hold the command buttons.
