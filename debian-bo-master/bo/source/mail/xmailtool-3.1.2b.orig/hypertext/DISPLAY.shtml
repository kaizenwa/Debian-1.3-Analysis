The DISPLAY Variable

The value of the DISPLAY variable is optained from either the DISPLAY environment variable at the time XMailTool is executed, the -display command line option, or it may be set in the .mailrc file. If the value is set in the .mailrc file, this value applies only to child prossesses and is not used by XMailTool.

Examples

csh:

setenv DISPLAY ringo:0

ksh/sh:

DISPLAY=ringo:0
export DISPLAY

.mailrc or .xmailtoolrc:

set DISPLAY="ringo:0.1"

Default

The value used by libXt
