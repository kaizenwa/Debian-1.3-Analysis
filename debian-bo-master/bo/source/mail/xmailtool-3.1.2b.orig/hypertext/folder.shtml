Setting the folder Variable

The folder variable is used to specify the path to the directory that is used to hold mailbox files. If a referenced file name starts with "+", XMailTool will substitute the value of "$folder/" before it uses it. If folder isn't set, then files with "+" will substitute the value of "$HOME/". If the folder variable isn't set, the Folders button will not be sensitive.

Examples

csh:

setenv folder mbox

ksh/sh:

folder=~/mail
export folder

.mailrc or .xmailtoolrc:

set folder="~joe/mbox"
set folder="mbox"
set folder="~/mbox"
set folder="~joe/mbox"
set folder="$HOME/mbox"

Default

folder has no default value.
