The ~/.signature file

The signature file allows the user to specify standardized ending for each outbound message. The contents will be copied to the end of the outbound message. Since the signature is considerd part of the message body, it can contain almost anything and there isn't a standard format.

Example

Have a day,

  @   @
   ( )     bobo
