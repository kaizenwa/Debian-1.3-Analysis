Imake Configuration

There are some important paramameters you need to set in the Imakefile. Before you start, you may want to make a copy of the original Imakefile. You may also want to read the documentation about Imake.

After modifying the Imakefile to suite your needs, you will execute one of the following commands:

    • xmkmf

      or

    • imake -DTOPDIR=. -DCURDIR=. -I$(X11CONF) [-DUseInstalled]
    If you are using an installed version of X11 and you use the second command line, you should use the -DUseInstalled option. In this case $(X11CONF) will probably be /usr/lib/X11/config. For Solaris 2.3 the X Consortium version of X11R5 is found in /opt/X11R5 and so $(X11CONF) is /opt/X11R5/lib/X11/config.

    If you are not using an installed version of X11, you should not use the -DUseInstalled option, and $(X11CONF) should be $(X11SRCROOT)/config (where $(X11SRCROOT) is the path to the root of the source distribution of X11.

  2. The command executed above will create a Makefile in the current directory. To make the Makefiles in the compface and utilities directories, you should type "make Makefiles."

Modifications made to the Imakefile in the XMailTool source root directory will require that you repeat step 1 above, but you shouldn't need to repeat step 2.

This is a list of the Imakefile configuration parameters you may need to change:

  • HAS_XFACE
  • SYS_LIBRARIES
  • HELP_DIR
  • HYPERTEXTDIR
  • MBOXDIR
  • MSENDER
  • PRINTER_CMD
  • NEED_DIRENT
  • NEED_TRUNCATE
  • NEED_MKSTEMP
  • NEED_STRSTR
  • NEED_STRERROR
  • NEED_CASECMP
  • NEED_SETENV
  • SPECIAL_FLAGS
  • INSTMAILFLAGS
  • HYPERTEXT
  • DEFURL
