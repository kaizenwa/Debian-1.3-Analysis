NEED_CASECMP

Some machines support a case insensitive version of strcmp(3) called strcasecmp(3). If your machine doesn't support this function, XMailTool will need the -DNEED_CASECMP flag when it is built. This will indicate that it should use it's own version of the function instead of expecting one in libc.

EXAMPLE

     NEED_CASECMP = -DNEED_CASECMP
XMailTool will use it's own version of strcasecmp(3). 

     NEED_CASECMP =
XMailTool will expect to find strcasecmp(3) in libc.
