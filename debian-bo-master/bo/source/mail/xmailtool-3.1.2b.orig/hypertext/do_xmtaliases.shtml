Setting the do_xmtaliases Variable

The do_xmtaliases variable is used to indicate that XMailTool should expand aliases it finds in the To:, Cc:, and Bcc: header lines. This can be usefull if the sendmail command does this for you.

Examples

set do_xmtaliases
set do_xmtaliases=1
set do_xmtaliases=true
set do_xmtaliases=yes
All of the above are equivalent and indicate that XMailTool should expand aliases. 
set do_xmtaliases=0
set do_xmtaliases=false
set do_xmtaliases=no
All of the above are equivalent and indicate that XMailTool should not expand aliases.

Default

set do_xmtaliases=1
