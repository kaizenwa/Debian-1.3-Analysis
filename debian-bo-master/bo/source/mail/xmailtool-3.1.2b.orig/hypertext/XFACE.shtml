X-Face Support in XMailTool

XMailTool recognizes and decodes the X-Face: header line if it exists in each message. The stuff after the X-Face: keyword is an encoded compressed 48x48 pixel bitmap. If the state_box variable is set, then the bitmap in the X-Face: header line will be displayed in the xface window. If a message doesn't contain an X-Face: header line, then XMailTool will display a bug looking thing in the xface window. You can include an X-Face: header line in your envelope file.

Creating an X-Face: header line

Follow these steps to create an X-Face: header line for your outbound messages:
  1. Get a 48x48 bitmap. This can be done using a scanner, drawing program, or copying an existing royalty free image.
  2. Using what ever means you have available, convert your 48x48 bitmap to either pbm or xbm format.

    Example

    icontopbm < myface.icon > myface.pbm
  3. Use either the pbmtoikon or xbm2ikon script to convert your 48x48 bitmap from pbm or xbm to ikon format.

    Example

    pbmtoikon < myface.pbm > myface.ikon
  4. Use compface to compress and encode the bitmap.

    Example

    compface myface.ikon
    This will produce output that looks similar to this: 
    #VkO-kk&'KNV~51;\~7WC\/1b.sA\3KIoogNW+K|EQN4AZ[#]cSBb1<iVza!G=3c+$uqP8Y
 0N](ft62cB$Q{Q?NRx{c{/VTCsqRfW}`V<dakecrK~[mvxe}QsxQ
  5. Copy this to your ~/.envelope file. You will need to add the X-Face: keyword to the begining of the first line.

    Example

    X-Face: #VkO-kk&'KNV~51;\~7WC\/1b.sA\3KIoogNW+K|EQN4AZ[#]cSBb1<iVza!G=3c+$uqP8Y
