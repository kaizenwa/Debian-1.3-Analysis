NEED_STRSTR

Some machines support a function named strstr(3) (of the form strstr(s1,s2)) which returns a pointer to the first occurance of string s2 in string s1. If your machine doesn't support this function, XMailTool will need the -DNEED_STRSTR flag when it is built. This will indicate that it should use it's own version of the function instead of expecting one in libc.

EXAMPLE

     NEED_STRSTR = -DNEED_STRSTR
XMailTool will use it's own version of strstr(3). 

     NEED_STRSTR =
XMailTool will expect to find strstr(3) in libc.
