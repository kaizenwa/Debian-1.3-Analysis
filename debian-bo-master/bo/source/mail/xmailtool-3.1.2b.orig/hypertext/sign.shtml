Setting the sign Variable

The sign/Sign variables are used to specify the string that gets inserted into an outbound message when you are using the internal editor and you type ~a/~A in the first column. If the value contains "\n", "\r", or "\t", these substrings are translated to NL, CR, and TAB respectivly.

Examples

set sign='Robert J. Kierski'
set Sign='Robert J. Kierski\nVisualization\t\t\tWhat a cool place\nCray Research'

Default

sign and Sign have no default value
