Using the {+|-}stateBox Command Line Option and Resource

This command line option and resource is similar to the state_box variable. It indicates that XMailTool should create a state toggle/command box at the top of the tool window. This window contains a radio group which controls the action taken when the user selects a message header in the header window. It also contains the X-Face image and the ShowDeleted, BodyOnly, and SaveCurrent toggles. If you prefer not to have the state box, you should use the +stateBox option or set the stateBox resource to false. The stateBox resource has a higher priority than the mailrc variable, and the command line option has a higher priority than the resource.

Examples

Command Line option

xmailtool +stateBox

Resource

XMailTool.stateBox: False

Default

By default XMailTool will create the state toggle/command box.
