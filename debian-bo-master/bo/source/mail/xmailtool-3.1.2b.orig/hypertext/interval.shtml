Setting the interval Variable

The interval variable is used to specify the period of time in seconds XMailTool should wait between checking the MAIL file for new mail. Setting this to a low value will mean that XMailTool will check more frequently which may cause more load on your system. A high value will mean that XMailTool will check less frequently which means you may not be notified right away when new mail arrives.

Examples

# 15 seconds
set interval=15
# 10 minutes
set interval=600
# overkill
set interval=1

Default

set interval=15
