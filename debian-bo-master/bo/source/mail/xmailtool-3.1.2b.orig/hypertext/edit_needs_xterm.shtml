Setting the edit_needs_xterm Variable

The edit_needs_xterm variable is used to indicate that the program specified by the EDITOR variable needs an xterm. If edit_needs_xterm is set, XMailTool will prepend "xterm -e" to the EDITOR string. The name of the xterm window will be "outbound_msg." If edit_needs_xterm is false, XMailTool assumes that EDITOR will create its own window.

Examples

set edit_needs_xterm
set edit_needs_xterm=1
set edit_needs_xterm=true
set edit_needs_xterm=yes
All of the above are equivalent and indicate that XMailTool should create an xterm. 
set edit_needs_xterm=0
set edit_needs_xterm=false
set edit_needs_xterm=no
All of the above are equivalent and indicate that XMailTool should not create an xterm.

Default

set edit_needs_xterm=1
