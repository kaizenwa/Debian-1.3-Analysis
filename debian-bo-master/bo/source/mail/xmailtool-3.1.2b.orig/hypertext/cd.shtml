Using the cd Command

The cd command is used in the mailrc file to change the working directory of the XMailTool process. The format of the cd command is as follows: 
cd [path]
The current working directory of the XMailTool process is displayed in the Directory: window. The current working directory of the XMailTool process will not change if XMailTool encounters an error when it tries to change directory (duh!).

Examples

cd
this is equivalent to 
cd $HOME
cd ~
