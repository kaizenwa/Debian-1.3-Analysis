Introduction

(Mirror Site Information is current as of April 30, 1997)

The following is a listing of the mirror sites--both WWW and FTP--that host Linux Gazette. Thanks to all the people who have graciously offered their services to make these sites possible.

A link and URL is provided for each site as well as an the e-mail address of a contact person. If you are having difficulties accessing a particular site, please send mail to the appropriate contact person. If a link is no longer functional, please mail me directly.

If you'd like to mirror Linux Gazette, files containing Linux Gazette issues are kept on our ftp site (link below) for downloading. Basic instructions for set up are in the README.mirrors file at that location. After you have set up your site, you can send us your mirror site information by using the form at the end of this page. If your browser doesn't support forms, you can simply e-mail your site details to gazette@ssc.com. Please include:

  • Subject: Mirror Site
  • URL for the web and/or ftp site
  • Name and e-mail address of the contact person
  • Optionally, your geographical location (in case it is not obvious from the domain name... :-)

If you have sent information about your mirror site and it doesn't yet appear on this list, please resend the URL. Mail has been known to get lost. Also, be careful that you type the URL and the e-mail address correctly.

Note that a tar+gzip'd version of Linux Gazette is available via anonymous ftp at indicated sites listed below. The naming designations are a clue as to the contents of each file:

  • LinuxGazette_xxx.tar.gz files include ALL documents from the first issue of the LG in July, 1995 to the most recent.
  • lg_issuexx.tar.gz files include ONLY the most recent set of documents for that issue, as well as any modifications to previous documents.

Enjoy!
