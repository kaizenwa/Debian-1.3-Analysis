UL

The <UL> represents a list of items -- typically rendered as a bulleted list.

The content of a <UL> element is a sequence of <LI> elements. For example: 

    <UL>
    <LI>First list item
    <LI>Second list item
     <p>second paragraph of second item
    <LI>Third list item
    </UL>
