Genscript vs. nenscript

$Id: genscript-vs-nenscript.html,v 1.1 1997/02/12 09:40:37 mtr Exp $

Differences

  • nenscript supports only latin1 and ascii character sets

    GNU Enscript supports latin1, ascii and many other character sets.

  • nenscript can't use proportional fonts; the only body font you can use is Courier.

    With GNU Enscript, you can use whatever body font you want. GNU Enscript can read AFM files which contain font metrics information so linebreaks and tab stops will print out correctly even if you print with a proportional font.

  • nenscript supports only A4 and Letter paper sizes.

    As a default GNU Enscript supports following output medias: A3, A4, A5, Letter, Legal, A4dj, Letterdj. Users can also add their own output media without re-compilation

Additional features in GNU Enscript

  • PostScript font downloading
  • Special escapes to inline EPS images and change body font on-the-fly.
  • Support for use-defined fancy headers.

Additional features in nenscript

For my best knowledge, GNU Enscript implements all the features which can be found from nenscript. If you know something which nenscript can do but genscript can't, please, let me know about it.
