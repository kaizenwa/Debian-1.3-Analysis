GNU Enscript vs. Adobe Enscript

$Id: genscript-vs-enscript.html,v 1.1 1997/02/12 09:40:30 mtr Exp $

Differences

  • Adobe Enscript supports only latin1 and ascii and ibmpc character sets

    GNU Enscript supports latin1, ascii, ibmpc and many other character sets.

Additional features in GNU Enscript

  • PostScript font downloading
  • special escapes to inline EPS images and change body font on-the-fly
  • support for use-defined fancy headers

Additional features in Adobe Enscript

  • support for PPD files
If you know something which Adobe Enscript can do but GNU Enscript can't, please, let me know about it. I can't implement it if I don't know about it.
