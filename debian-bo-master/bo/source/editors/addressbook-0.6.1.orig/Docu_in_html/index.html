Addressbook is a program to manage your addresses, phone numbers, e-mailaddresses and everything which is now written in your rolodex.

Actual Version: 0.6.1

Addressbook 0.6.1 has been released the 02.05.1996. It fixes some minor bugs of 0.6 and adds some new features and configurable options. Feel free to update.

Addressbook is copyrighted by Clemens Durka, but available under the GNU GPL. If you decide to use the program, I would appreciate if you could send me back the registation file included in the package. It is free, but it helps me to see, how many people use it and if it is worth to continue the development.

To upload addressbook via http push here (with netscape push left mousebutton and use save as): addressbook-0.6.1.tar.gz (156 KBytes)

If this doesn't work, you can get the last major release 0.6 via ftp at the following sites and their mirrors:
ftp://ftp.leo.org/pub/comp/os/linux/local/apps/addressbook-0.6.tar.gz
ftp://sunsite.unc.edu/pub/Linux/X11/xapps/rolodex/addressbook-0.6.tar.gz (not there yet)
ftp://ftp.aud.alcatel.com/tcl/code/addressbook-0.6.tar.gz (not there yet)

I wrote Addressbook during a project at LEA (Lehrstuhl für Effiziente Algorithmen) at the Technische Universität München. Thanks to Prof. Dr. E. Mayr for the support.
