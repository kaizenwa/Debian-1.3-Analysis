Simple editor for X (SeX)

SeX?

SeX is the Simple editor for X. See the manual for more information.

The SeX files

I use the Debian GNU/Linux system, so I provide pre-packaged binaries for Debian. For other systems you will need to download the source and compile it yourself.

To compile SeX, you need my Publib library as well. Install it first.

Bugs

The sources include a file called BUGS, which contains a list of known bugs and misfeatures. There are lots of small ones. I don't know of any big ones.
