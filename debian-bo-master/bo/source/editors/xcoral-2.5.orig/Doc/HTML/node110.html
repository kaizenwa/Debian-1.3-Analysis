Assignment

  • incompatible types in assignment, <type1> <> <type2> when the assigned value is incorrect.

    (incompatible types in assignment for i int <> char*)

  • incompatible types in array assignment, <type1> <> <type2>

  • illegal assignment <var\_name> = .. when the variable (local or global) is an array.

  • illegal assignment <var\_name>++ or illegal assignment <var\_name>-/- when the variable (local or global) is an array.

  • illegal assignment when you attempt to assign a constant:

  • illegal assignment (*& = ...) andillegal assignment (*(&)++/-/-)
