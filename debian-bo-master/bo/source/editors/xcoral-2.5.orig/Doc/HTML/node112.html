Initialization

  • illegal array initializer for <var\_name>

  • incompatible initial value type for <var\_name> <type1> <> <type2> when the initialization value does not have a good type.

    int i = "1";/*incompatible initial value type, i int <> char* */

  • array initialization not yet implemented

    int tab[2] = {1, 2};

  • {..} illegal initializer when using brace list to initialize something else than an array.

    int i = {123};
