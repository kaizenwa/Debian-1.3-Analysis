Browser

  

A browser is available, in C or C++ mode, to navigate through C functions and C++ class hierarchies and methods. The names of files, C functions, classes, parents, children and methods are listed in subwindows of the control browser window. An object selected by one mouse click in one of these subwindows is viewed in a 'Visit Window'. A double click opens an edition area and loads the current selected object (see §3.1 page ).
