Xcoral interface

 

The interface between Smac and Xcoral is composed of a large list of built-in functions. With them you can customize your Xcoral environment by adding your own functions, for instance to define modes (like the C-mode, see §5.24 page ), to automatically indent your code files, or other features (see also the cmd.sc file description §5.10 page ).
