Colors

 

color_area

  void color_area(int start, int end, char *colorname)

Colors text region from position start to position end with color colorname (see the standard X11 file rgb.txt). This function can be used with the Smac regexp interface to color strings of your own modes (see color.sc §5.11 page ).

remove_colors

  void remove_colors()

Removes all colors in the current buffer.

monochrome

  int monochrome()

Returns 1 if your display is monochrome, else 0.
