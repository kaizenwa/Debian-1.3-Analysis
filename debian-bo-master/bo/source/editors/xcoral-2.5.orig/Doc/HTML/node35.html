Writing new functions

 

The following simple examples can be written and tested on-line. First you must open a new window to write these functions (see example.sc). Before evaluating one or all functions with Esc e, you must select it or them in a region (see §2.9 page ). To execute a function use Ctrl-x Ctrl-e then enter function_name(); and hit return.
