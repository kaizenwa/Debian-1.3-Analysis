latex-macros.sc

 

Latex mode has been provided with a latex macros menu. This menu allows you to automatically insert latex macros in your current file.

For example, to insert:

double click on the items Other environments.. and Minipage in Latex style and macros from the menu Misc. Macros inserted are automatically indented and colored. Some macros like Font style can be used with a predefined region.
