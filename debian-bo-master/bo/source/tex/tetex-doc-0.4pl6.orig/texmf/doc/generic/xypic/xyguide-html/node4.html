Xy-pic User's Guide

1.1 Loading

The Xy-pic setup used in this guide is loaded by inserting the lines

\input xy
\xyoption{all}

in the definitions part of your document. If you wish to load only the features you use, or you wish to use non-standard facilities like the v2 backwards compatibility mode or the ps PostScript backend then this is also possible as described in the reference manual [8].
