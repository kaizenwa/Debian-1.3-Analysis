Getting Xy-pic

The easiest way to retrieve the latest version of Xy-pic is through the Xy-pic home page [5] on the World Wide Web; it is also available by Internet ftp from ftp://ftp.diku.dk/diku/users/kris/TeX/ as well as ftp://ftp.mpce.mq.edu.au/pub/maths/TeX/ in files starting with xy. Finally, Xy-pic has been contributed to the CTAN archives where it is located in the directory macros/generic/diagrams/xypic.
